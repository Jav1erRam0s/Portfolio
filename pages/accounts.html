Accounts

Accounts es un sistema que permite organizar cuentas virtuales que tanto abundan y no sabemos como guardar sin usar una misma contraseña para todo. Es un sistema fácil de usar y muy practico para cualquiera.
Una cosa a considerar, es que este sistema no se baso en ningún modelo de aplicación de guardado de contraseñas, fue pensado de manera personal, cumpliendo lo que yo creí necesario para mi mismo.
A continuación se describen las funciones.
  1. ABM cuenta: Permite registrar, editar y eliminar cuentas virtuales.
  2. Importar y exportar BD: Permite realizar un backup de la base de datos.
  3. Descargar cuentas: Guardar todas las cuentas registrardas en un formato pdf, ordenado y practico para tenerlo todo a mano.
  4. Busqueda: Permite buscar una cuenta sobre el stack.
  5. Configurar password: El sistema es seguro, dado que la informacion esta encriptada, por ende necesita una llave para poder acceder a la informacion. Esta funcionalidad permite cambiar la llave de la aplicacion.
  6. Redireccionar a la cuenta: Permite ir al login page de la cuenta a acceder, haciendo doble click o Enter sobre la cuenta misma.
  7. Copiar a portapapeles: Cualquier celda se puede copiar a portapeles solo haciendo click sobre ella. Esto hace mas rapido el proceso de login.
A continuación comparto la aplicación ya que puede ser muy útil para tod@s. Una aclaración importante es que admin es la contraseña por default al momento que se instala. Por ultimo tenga en cuenta que al momento de exportar la BD también lo hace con la contraseña (hasheada) que uso hasta ese momento y si lo importa para acceder debe hacelo con esa contraseña.
